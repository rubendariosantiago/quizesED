Examen de Ecuaciones Diferenciales

Formato Requerido

Ejemplos válidos:

  • (2*cos(3*x) + sin(3*x))/5 + 3*exp(-2*x)
  • (x^2 + 1)/(x - 1) + exp(-x)
  • 3*sin(2*x) - 4*cos(x/2)

Reglas estrictas:

  • Usa * para multiplicación (2*x)
  • Usa paréntesis para agrupar: (a+b)/(c+d)
  • Funciones: cos(x), sin(x), exp(x)
  • No uses espacios
  • Para fracciones: (numerador)/(denominador)
