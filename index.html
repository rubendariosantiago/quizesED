Examen de Ecuaciones Diferenciales

Formato Requerido

Ejemplos válidos:

  • 2*cos(3*x) + exp(-2*x)
  • (3*cos(2*x) + 4*sin(2*x))/5 + 2*exp(-3*x)
  • 5 + (x-1)*log(x+1)

Requisitos:

  • Usa siempre * para multiplicación
  • Funciones deben llevar paréntesis: cos(x) no cos x
  • No uses espacios
  • Usa exp(x) para exponenciales
  • Usa ^ o ** para potencias
